2) assets/c9_main_banner.png 파일을 GitHub 저장소의 assets/c9_main_banner.png 위치로 업로드/커밋/푸시하세요.

3) make_ygosu_c9_dashboard.py 실행

   두 파일이 생성됩니다.
4) 기존 자동업로드는 auto_edit_ygosu_c9_dashboard.py 그대로 사용하면 됩니다.
