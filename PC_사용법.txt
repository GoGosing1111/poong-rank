PC 자동 판독형 위장팬덤 셀프인증 사용법

1. GitHub poong-rank 최상단에 soop_fandom_verify.js를 업로드/push 합니다.
2. make_ygosu_c9_dashboard.py를 덮어쓴 뒤 현황판을 재생성/와고 자동수정합니다.
3. 와고 현황판에서 [자동 인증 코드 복사]를 누릅니다.
4. [시청기록 확인]을 눌러 SOOP 시청기록 페이지를 엽니다.
5. 주소창에 javascript: 입력 후 복사된 코드를 붙여넣고 Enter 합니다.
6. 와이고수 닉네임 입력 후 생성된 PNG를 저장해서 해명자료로 사용합니다.

판독 대상: 염보성 / 비제이 케이
