🚨 위장팬덤 셀프인증 PC버전 사용법

[설치 1회]
1. soop_fandom_verify.js 파일을 GitHub poong-rank 저장소 최상단에 업로드/푸시한다.
   예: https://keyman1335-maker.github.io/poong-rank/soop_fandom_verify.js

2. bookmarklet_pc.txt 안의 javascript: 코드 전체를 복사한다.

3. 크롬 즐겨찾기 하나를 만들고 URL 주소에 javascript: 코드를 붙여넣는다.
   이름 예: 위장팬덤 셀프인증

[사용]
1. 현황판의 🚨 위장팬덤 셀프인증 > 시청기록 확인하기 클릭
2. SOOP 로그인 상태로 시청기록 페이지 접속
3. 크롬 즐겨찾기에서 '위장팬덤 셀프인증' 클릭
4. 와이고수 닉네임 입력
5. 인증 이미지가 뜨면 PNG 저장
6. 저장한 이미지를 와이고수 해명자료로 첨부

[판정 대상]
- 염보성
- 비제이 케이 / BJ케이 / 케이

[주의]
- PC 크롬 기준으로 먼저 테스트하는 버전이다.
- 페이지에 표시된 본인 시청기록 DOM에서 대상 BJ 문자열을 찾는 방식이다.
- SOOP 페이지 구조가 바뀌면 JS 수정이 필요할 수 있다.
